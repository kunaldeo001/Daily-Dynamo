# Daily Dynamo ⚡️

Daily Dynamo is a whimsical and interactive day planner designed to spark joy and boost productivity through AI-driven insights and focused tools.

## Features

- **AI Daily Spark**: Start your day with a creative challenge tailored to your current mood (Motivated, Happy, Meh, or Tired).
- **Focus Dynamo**: A built-in Pomodoro timer to help you maintain your flow and balance work with quick recharges.
- **Dynamo Central**: Organize your tasks into three core categories: **Productive**, **Self-Care**, and **Whimsical**.
- **AI Task Breakdown**: Use the Productivity Wizard to break down intimidating tasks into three fun, actionable sub-steps.
- **Daily Momentum**: Visualize your achievements with a real-time productivity dashboard and completion stats.
- **Secure Sync**: All your tasks and sparks are securely stored and synced via Firebase, allowing you to pick up where you left off.

## Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) & [ShadCN UI](https://ui.shadcn.com/)
- **AI**: [Google Genkit](https://github.com/firebase/genkit) (Gemini 2.5 Flash)
- **Backend**: [Firebase](https://firebase.google.com/) (Authentication & Firestore)
- **Charts**: [Recharts](https://recharts.org/)
- **Icons**: [Lucide React](https://lucide.dev/)

## Getting Started

1. **Firebase Setup**: Ensure your Firebase project is configured in `src/firebase/config.ts`.
2. **AI Configuration**: Set up your Google Generative AI API key in your environment.
3. **Install Dependencies**: Run `npm install`.
4. **Development**: Run `npm run dev` to start the local development server.
